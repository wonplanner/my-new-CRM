
# 🛡️ 인슈어플래너 CRM (Insure Planner) v2.5

보험 설계사를 위한 iOS 스타일의 프리미엄 고객 관리(CRM) 솔루션입니다. 복잡한 설치 없이 브라우저에서 즉시 고객 데이터를 관리하고 Google Gemini AI를 통한 비즈니스 인사이트를 얻으세요.

## 🚀 핵심 가치
- **Privacy First**: 모든 데이터는 서버가 아닌 설계사의 브라우저 로컬 스토리지에 암호화되어 저장됩니다.
- **AI Intelligence**: Google Gemini-3-Flash API를 활용하여 고객 현황 분석 및 오늘의 우선순위를 제안합니다.
- **Visual Networking**: 가족, 지인 등 고객 간의 복잡한 관계를 시각적 지도로 관리합니다.

## ✨ 주요 기능
- **AI 비즈니스 분석**: 등록된 고객 데이터를 분석하여 매일 아침 맞춤형 리포트 생성.
- **정밀 필터링**: 유지/해지/잠재 상태별, 생일 월별, 가입 상품 태그별 멀티 필터링.
- **단체 SMS 최적화**: iOS와 Android를 자동 인식하여 단체 발송 링크 생성 및 템플릿 지원.
- **데이터 센터**: 전체 데이터를 CSV(엑셀)로 백업하거나 PDF 리스트로 즉시 출력.
- **iOS 디자인**: 애플의 디자인 시스템을 따르는 직관적이고 미려한 UI.

## 🛠️ 기술 스택
- **Framework**: React 19 (ESM)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **AI Engine**: Google GenAI (Gemini-3-Flash-Preview)
- **Deployment**: Vercel

## 📥 시작하기 (개발자용)
1. 리포지토리를 클론합니다: `git clone https://github.com/your-id/insure-planner.git`
2. 환경 변수를 설정합니다: `.env` 파일 생성 후 `API_KEY=your_gemini_api_key` 입력
3. 의존성을 설치합니다: `npm install`
4. 로컬에서 실행합니다: `npm start`
